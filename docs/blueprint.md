# **App Name**: Umefog Webbplats

## Core Features:

- Service Presentation: Display comprehensive service offerings with descriptions.
- Portfolio Showcase: Present past projects to demonstrate expertise and capabilities.
- Contact Form: Enable users to directly contact the company for inquiries or service requests.
- Svenska Localization: Full website content in Svenska with adherence to Svenska grammar rules.

## Style Guidelines:

- Primary color: Light Orange (#f97316) for calls to action.
- Background color: Very Light Slate (#f8fafc) for a neutral, professional backdrop.
- Accent color: Pale Slate (#cbd5e1) to create separation for the content sections on the page.
- Slate Color: #334155 as the main dark color
- Headline font: 'Playfair' (serif) for titles and short text amounts. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans' (sans-serif) for descriptions and paragraph texts. Note: currently only Google Fonts are supported.
- Simple, professional icons to represent services and sections.
- Clean and modern layout leveraging Tailwind CSS for responsiveness.
